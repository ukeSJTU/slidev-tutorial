# 编辑器设置 (VS Code) 🛠️

推荐使用 VS Code 并安装官方 **Slidev** 扩展 (`antfu.slidev`) 以获得最佳体验。

**主要功能:**

-   **侧边栏预览:** 实时查看幻灯片效果。
-   **幻灯片树视图:** 方便导航和组织幻灯片。
-   **幻灯片排序:** 拖放调整顺序。
-   **一键启动服务:** 快速启动开发服务器。
-   **预览同步:** 编辑器光标与预览联动。

<br>

**使用方法:**

1.  安装 `antfu.slidev` 扩展。
2.  点击 VS Code 活动栏的 Slidev 图标打开面板。
3.  在面板中管理和预览你的项目。
